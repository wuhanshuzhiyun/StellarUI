# 安装
### NPM配置

```javascript
npm config set registry http://172.16.109.59:8010/repository/npm-cnpm/
```

### NPM安装

```
npm install stellar-ui -S
```

### 引入配置
```json
// pages.json
{
	"easycom": {
		"autoscan": true,
		"custom": {
			"ste-(.*)": "stellar-ui/components/ste-$1/ste-$1.vue"
		}
	}
}
```

### 使用
```html
<ste-button type="primary" size="mini" @click="onClick">点击</ste-button>
```
