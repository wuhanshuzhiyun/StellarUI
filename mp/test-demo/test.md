### API
#### 组件属性(Props)

| 属性名   | 说明                                      | 类型    | 默认值 | 可选值                                    |
| -------- | ----------------------------------------- | ------- | ------ | ----------------------------------------- |
| 属性名 | 属性说明                      | 属性类型(Boolean) | 默认值(true)   | 列举可选值(true / false)                              |

#### 组件事件(Events)

| 事件名 | 说明             | 类型    | 默认值 | 可选值 |
| ------ | ---------------- | ------- | ------ | ------ |
| 事件名(click)  | 事件说明 | 事件类型 | -      | -      |

#### 组件插槽(Solts)

| 名称    | 说明     |
| ------- | -------- |
| 插槽名称 | 插槽说明 |
